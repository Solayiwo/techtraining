Brief summary about GitHub

GitHub is a web-based code hosting platform for version control and collaboration. 
It offers all of the distributed revision control and source code management (SCM) 
functionality of Git as well as adding its own features. It allow group of developer
to work together on projects from anywhere.

Features of github
It allows users to create a new directory.
It allows users to Create and manage a new branch 
Users are allowed make and commit changes on a file under a particular repository	
It allows user to show diffs, or differences, of the content from both branches by using pull request.
It also allow user to open and merged a pull request.
Github enable a group of developer to collaborate on a particular project.
